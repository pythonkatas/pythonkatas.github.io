4. Katas

Чтобы увидеть полное Решение перейдите на GitHub. Смотрите ниже↓

Решение:

  1. Создаем класс
  2. Создаем метод конструктор в класс который принимает одну переменную(строку). В самом конструкторе определяем четыре переменных(нашу строку, лист из гласных и два файла в которые будем записывать наши строки)
  3. Создаем функцию для записи строки в файл, которая принимает строку и название файла
  4. Создаем функцию для преобразования строки в строку без гласных
  5. В самой функции:
    • Записываем строку в новую переменную.
    • Пробегаем по всей переменой и удаляем гласные.
    • Записываем нашу строку в файл.
    • Возвращаем нашу строку.
  6. Создаем функцию для преобразования строки в строку без согласных, то же алгоритм действий что из предыдущий функцией.
  7. Создаем метод для принта класса, котрый будет выводит строку, которая нам нужна по условмию.

На этом и все.
