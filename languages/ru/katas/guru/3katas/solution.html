3. Katas

Чтобы увидеть полное Решение перейдите на GitHub. Смотрите ниже↓

Решение:

  1. Нулевая функция:
    • Создаем функцию с одной переменной для входа(количество товаров для записи в файл), которая записывать товары в файл.
    • В самой функции:
      • Открываем файл.
      • Создаем цикл в котором:
        • Вводим название продукта, страну и объем.
        • Добавляем три переменные в список.
        • Конвертируем список в строку для записи в файл.
        • Добавляем строку в файл.
      • Закрываем файл.
      • Возвращаем название файла в который все записали для дальнейших действий с ним.
  2. Добавляем переменную в которую после запуска программы будет записывать количество продуктов для добавление в файл.
  3. Создаем еще одну переменную в которой вызываем нашу функцию с предыдущей переменной.

На этом и все.
